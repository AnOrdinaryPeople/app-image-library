# app-image-library

## Project Setup

```sh
# setup variables VITE_IMGBB_API_URL and VITE_IMGBB_API_KEY from https://api.imgbb.com
cp .env.example .env

yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
